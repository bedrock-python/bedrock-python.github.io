# Tools

Standalone developer tools built with Python. No integration into your codebase required — install and run.

---

## mr-review

AI-powered merge request review that runs locally via Docker. Connects to GitLab, GitHub, Gitea, Forgejo, or Bitbucket and works with Claude, OpenAI, or any OpenAI-compatible model. Reviews the diff in stages (Brief → Dispatch → Polish → Post) and lets you approve inline comments before posting back to the MR.

[![Docs](https://img.shields.io/badge/docs-github.io-blue)](https://bedrock-python.github.io/mr-review/)
[![GitHub](https://img.shields.io/badge/source-github-181717?logo=github)](https://github.com/bedrock-python/mr-review)

## mattermind

CLI tool for asking natural-language questions about your Mattermost workspace. Uses LLM tool-calling with full-text search, agentic loops, and a rich terminal UI. Supports SSO-compatible authentication, token budget protection, and JSON output mode for scripting.

[![Docs](https://img.shields.io/badge/docs-github.io-blue)](https://bedrock-python.github.io/mattermind/)
